<h2>VSClearType</h2>

Willkommen bei ClearType für Visual Studio 2022

## Übersicht

Bei dieser Visual Studio Extension handelt es sich um eine Erweiterung, die das Textrendering in der Visual Studio IDE und all ihrer Teilkomponenten anpasst.

Auf der Einstellungsseite dieser Erweiterung, die sich in den Visual Studio Einstellungen unter "ClearType" befindet, können die Methoden für das Text-Rendering, das -Formatting und das -Hinting ausgewählt werden.

Die Einstellungen, die in auf dieser Seite gewählt werden, werden exemplarisch direkt auf die gesamte Visual Studio IDE angewendet. Sobald [OK] gewählt wird, werden die Änderungen übernommen und in die Windows Registry unter "Software\VSExtensions\ClearType" geschrieben.

Wenn [Cancel] gewählt wird, werden alle Änderungen verworfen und der ursprüngliche Zustand wiederhergestellt.

Angenehme Arbeit mit ClearType-Unterstützung in der Visual Studio IDE...
